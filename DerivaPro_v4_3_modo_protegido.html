Configura el análisis

Usa x como variable, ^ para potencias y paréntesis para denominadores completos. Ejemplo: 1/(x*exp(8)). La precedencia de operaciones se conserva tal como la escribas.

Atajo: presiona Ctrl + Enter para calcular.
